# Poster AR Preview

모바일 웹에서 포스터 이미지를 실제 공간에 띄워 구매 전 크기와 분위기를 확인하는 AR 프리뷰 앱입니다.

사용자는 카메라를 켠 뒤 원하는 벽 앞에 서서 화면 중앙을 기준으로 포스터 이미지를 배치하고, 크기를 조절해 실제 벽과 주변 가구/기물의 비율에 맞춰볼 수 있습니다.

배치된 포스터는 카메라를 따라다니지 않고 월드 좌표에 고정됩니다.

---

## 목적

이 프로젝트의 목적은 포스터 구매 전 사용자가 자신의 실제 공간에서 이미지를 미리 확인할 수 있게 하는 것입니다.

핵심 목표는 다음과 같습니다.

- 모바일 웹에서 앱 설치 없이 AR 프리뷰 제공
- 사용자가 원하는 벽 앞에서 포스터 이미지를 화면 중앙에 배치
- 포스터 크기를 사용자가 직접 조절
- 배치된 포스터가 공간 좌표에 고정되어 사용자가 움직여도 그 자리에 유지
- 향후 위치, 회전, 기울기 등 미세 조정 기능 확장 가능

---

## 핵심 UX

1. 사용자가 웹앱에 접속합니다.
2. 카메라 권한을 허용합니다.
3. 포스터를 놓고 싶은 벽 앞에 섭니다.
4. 화면 중앙 가이드를 원하는 위치에 맞춥니다.
5. `이미지 배치` 버튼을 누릅니다.
6. 포스터 이미지가 AR 공간에 생성됩니다.
7. `작게`, `크게`, `기본크기` 버튼으로 크기를 조절합니다.
8. 필요하면 `위`, `아래`, `다시 배치` 버튼으로 위치를 조정합니다.
9. 사용자가 뒤로 물러나거나 옆으로 이동해도 포스터는 해당 공간 좌표에 고정됩니다.

---

## 현재 구현 범위

현재 버전은 자동 벽면 감지보다 안정적인 사용자 배치 방식을 우선합니다.

지원 기능:

- 8th Wall 기반 모바일 Web AR
- A-Frame scene
- 상품 선택
- 화면 중앙 기준 이미지 배치
- 월드 좌표 고정
- 크기 조절
- 위/아래 위치 조절
- 다시 배치
- 트래킹 기준 재계산

현재 제외된 기능:

- 자동 벽면 감지
- WebXR plane-detection
- WebXR hit-test 기반 거리 측정
- 좌우 이동
- 좌우 회전
- 기울기 보정
- 저장/공유 기능
